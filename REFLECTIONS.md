# CW-04 Reflection Responses

1) Why choose class components here?

- The assignment focuses on practicing `class` components and `this.setState`.

2) How does state flow between components?

- `Counter` and `FilteredList` manage their own internal state. `App` passes data
  as props to `List` and `FilteredList`, showing unidirectional data flow.

3) Where is filtering useful in real apps?

- Search and dropdown filtering are common in e-commerce product lists,
  admin dashboards, and any UI that needs to narrow large datasets.
